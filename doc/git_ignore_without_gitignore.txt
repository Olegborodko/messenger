Используйте локальные правила игнорирования: Git поддерживает локальные файлы .gitignore для каждого пользователя. Эти файлы хранятся в домашнем каталоге пользователя и позволяют игнорировать файлы на локальном уровне, не затрагивая репозиторий в целом.
Создайте файл .git/info/exclude в корне вашего репозитория. Этот файл действует аналогично .gitignore, но он применяется только к конкретному репозиторию и не будет виден другим разработчикам.
